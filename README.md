# MyFinances - Smart Budget Manager

A modern, sleek, and functional budget management system built with Vanilla HTML, CSS, and JavaScript.

## Features

- **Dynamic Dashboard**: Real-time updates of balance, income, and expenses.
- **Visual Analytics**: Interactive doughnut charts for expense breakdown by category.
- **Transaction History**: Track your spending with a clean, searchable list.
- **Data Persistence**: All your data is saved locally via `localStorage`.
- **Responsive Design**: fully functional on desktop and mobile devices.
- **Dark/Light Mode**: Toggle between themes for comfort.

## Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Custom properties, Flexbox, Grid, and Animations.
- **JavaScript (ES6)**: Core logic and data management.
- **Chart.js**: Data visualization.
- **Lucide Icons**: Beautiful, consistent iconography.
- **Google Fonts**: Inter and Outfit.

## Setup

1. Clone the repository.
2. Open `index.html` in your favorite browser.
3. Start managing your finances!
